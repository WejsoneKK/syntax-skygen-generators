## Commands

(Placeholder: List any specific commands provided by your plugin/generators. If it's purely an API or configuration, state that.)

*   `/syntaxgenerator stwórz <nazwa>` - Tworzy generator o podanej nazwie
*   `/syntaxgenerator usuń <nazwa>` - Usuwa generator o podanej nazwie
*   `/syntaxgenerator dodajitem <nazwa> <ilość>` - Dodaje określoną ilość itemków trzymanych w rękach dla podanego generatora
*   `/syntaxgenerator usunitem <nazwa> <id-itemu>` - Usuwa przedmiot z danego generatora
*   `/syntaxgenerator personal <nick> <moc> <czas>` - Nadaje graczowi genboosta o podanej mocy na określony czas
*   `/syntaxgenerator global <moc> <czas>` - Nadaje globalnego genboosta o podanej mocy na określony czas.
*   `/syntaxgenerator info <nazwa>` - Pobiera informacje o danym generatorze.

## Building the Project

This project uses Gradle. To build it, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/syntax-skygen-generators.git
    cd syntax-skygen-generators
    ```
2.  **Build with Gradle:**
    ```bash
    ./gradlew build
    ```
    (On Windows, use `gradlew.bat build`)

    The compiled JAR file will be located in the `build/libs/` directory.
